# Search

Agaric ships with a built-in **full-text search** that scans every block and page title in the current space. Open the panel from the sidebar (or with the assigned keyboard shortcut — see [docs/UX.md](UX.md) for current key bindings), type a query of three or more characters, and results stream in grouped by page. Matches inside snippets are highlighted; the page header doubles as a navigation target so you can jump straight to the parent. The same panel exposes the legacy `+ Page` and `+ Tag` filter chips while the inline filter syntax is in flight.

For an in-app reference, click the `?` button in the search toolbar — it opens a help dialog mirroring the section list below.
